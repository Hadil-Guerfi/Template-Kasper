HTML Tables

What's Html Tables

HTML tables allow web developers to arrange data into rows and columns.

Example:


Define an HTML Table

A table in HTML consists of table cells inside rows and columns

Example:

A simple HTML table:

looks like:
Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Table Cells

Each table cell is defined by a <td> and a </td> tag.
Everything between <td> and </td> are the content of the table cell.

Example:


looks like:
Emil Tobias Linus

Note:

table data elements are the data containers of the table. They can contain all sorts of HTML elements; text, images, lists, other tables, etc.

Table Rows

Each table row starts with a <tr> and end with a </tr> tag.

Example:


looks like:
Emil Tobias Linus
16 14 10

Table Headers

Sometimes you want your cells to be headers, in those cases use the <th> tag instead of the <td> tag:

Example


looks like:
Person 1 Person 2 Person 3
Emil Tobias Linus
16 14 10

HTML Table Tags

HTML Table Colspan & Rowspan

HTML tables can have cells that spans over multiple rows and/or columns.
like this:

HTML Table - Colspan

To make a cell span over multiple columns, use the colspan attribute:
Example:

Looks like:
Name Age
Jill Smith 43
Eve Jackson 57
Note:
To make a cell span over multiple rows, use the rowspan attribute

HTML Table - Rowspan

To make a cell span over multiple rows, use the rowspan attribute
Example:

Looks like:
Name Jill
Phone 555-1234
555-8745
Note:
The value of the rowspan attribute represents the number of rows to span.
